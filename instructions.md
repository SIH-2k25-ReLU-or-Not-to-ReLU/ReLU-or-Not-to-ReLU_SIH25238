# InsightGuard - Folder Structure Overview

**Attack_Scripts/** - Contains APT attack simulation scripts covering the full kill chain (reconnaissance, privilege escalation, persistence, exfiltration) for testing detection capabilities.

**Documentation/** - Stores project documentation, research papers, and reference materials related to APT detection and ELK Stack implementation.

**Implementation_Manual/** - Comprehensive guide with step-by-step setup instructions for ELK infrastructure, attack scenarios, detection rules, and MITRE ATT&CK mappings.

**Machine-Learning/** - Implements ML-driven anomaly detection using Deep Reinforcement Learning (DRL) agents and synthetic data generation for behavioral threat analysis.

**Test-Exploitation/** - Contains proof-of-concept scripts for testing data exfiltration methods (DNS and HTTP) to validate detection rules.
